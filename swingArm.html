Tilting Urban Mobility Vehicle

What?

  • Create a generative design for the swing arm used in motorbikes which can be 3D printed.

How?

  • Used generative design module in Fusion 360 to create the design.
  • Used editing features to clean up the model to make it suitable for 3D printing.

Results

  • Weight reduction of 40 percent from the conventional swing arm design.
